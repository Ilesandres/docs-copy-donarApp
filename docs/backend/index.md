# Backend Overview

The DonarApp backend is built using [NestJS](https://nestjs.com/), a progressive Node.js framework for building efficient, reliable and scalable server-side applications.

## Key Technologies

*   **NestJS**: Core framework.
*   **TypeScript**: Language.
*   **TypeORM**  ORM for database interaction.
*   **PostgreSQL/MySQL**  Database.

## Architecture

The backend follows the standard NestJS modular architecture:

*   **Modules**: Encapsulate related code (controllers, services, etc.).
*   **Controllers**: Handle incoming HTTP requests and return responses.
*   **Services**: Contain the business logic.
*   **Entities**: Define the data structure and database schema.
